# Contoh Dokumen Bisnis

Ini adalah contoh file yang bisa dibaca dan dimodifikasi oleh AI Agent.

## Cara Menggunakan

1. Simpan file Anda di folder ini
2. AI Agent dapat membaca, menganalisis, dan memodifikasi file
3. File yang diizinkan: .txt, .md, .json, .csv, .py, .js, .html, dll

## Tips

- Gunakan fitur "File Manager" di Dashboard untuk melihat semua file
- Upload file baru via Chatbot dengan tombol 📎
- AI akan otomatis membaca konteks file saat Anda bertanya
