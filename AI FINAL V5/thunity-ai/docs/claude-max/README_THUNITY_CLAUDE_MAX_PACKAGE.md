# THUNITY CLAUDE MAX PACKAGE NOTE

This package contains supporting instruction files for moving Thunity AI engineering work to Claude Max 20x.

Recommended setup order:

1. Add the latest Thunity project folder/ZIP.
2. Paste the YAML operating handoff into the Claude project instructions.
3. Add these files:
   - `CLAUDE_MAX_START_HERE.md`
   - `UI_REFERENCE_RULES.md`
   - `CLAUDE_MAX_FIRST_TASK_ORIENTATION.md`
4. Add logo/color/UI reference assets only if they are intentional and curated.
5. Start Claude Max with the orientation check, not implementation.

Important:
Do not ask Claude Max to “continue” or “fix everything.”
Start with orientation, then W14N QA gate, then tiny QA-fix patches or a new-domain scout.
